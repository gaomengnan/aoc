* K-th element of two Arrays

给定两个已排序的数组 a[] 和 b[] 以及一个元素 k，任务是找到合并排序数组中第 k 个位置的元素

** 思路
*** 嵌套循环

*** 双指针
#+begin_src python
    class Solution:
        def kthElement(self, a, b, k):
        # code here
            ans = []
            x = 0
            y = 0
            while x < len(a) and y < len(b):
                # if x > 0 and  a[x] == a[x-1]:
                #     x +=1
                #     continue

                # if y > 0 and b[y] == b[y-1]:
                #     y +=1
                #     continue

                if a[x] < b[y]:
                    ans.append(a[x])
                    x += 1
                elif a[x] == b[y]:
                    ans.append(a[x])
                    ans.append(b[y])
                    x += 1
                    y += 1
                else:
                    ans.append(b[y])
                    y += 1


            ans.extend(a[x:])
            ans.extend(b[y:])

        return ans[k - 1]
