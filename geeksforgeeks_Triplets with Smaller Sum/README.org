* 问题

** Triplets with Smaller Sum

*** 描述

给定一个大小为 n 的数组 arr[]（数组元素互不相同），以及一个整数 sum。
任务是找到满足条件的三元组 (i, j, k) 的个数，其中：

    - i < j < k
    - arr[i] + arr[j] + arr[k] < sum


*** 解法

**** 排序+ 双指针

#+begin_src python

class Solution:
    def countTriplets(self, n, sum, arr):
        ans = 0
        arr.sort()
        for i in range(n):
            x = i + 1
            y = n - 1

            while x < y:
                toSum = arr[i] + arr[x] + arr[y]

                if toSum >= sum:
                    y -= 1
                else:
                    ans += y - x
                    x += 1
        return ans

#+end_src
